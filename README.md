# Basic Web Development - Library Management System

Ini tugas kuliah untuk membuat website sederhana sistem manajemen perpustakaan menggunakan HTML.

## Deskripsi
Project ini berisi beberapa halaman statis untuk simulasi alur dasar perpustakaan:
- Login
- Dashboard
- Daftar buku
- Data member
- Form peminjaman buku
- Form pengembalian buku

## Halaman
- login.html
- index.html
- books.html
- members.html
- borrow.html
- return.html

## Fitur
- Navigasi antar halaman
- Form login sederhana
- Form peminjaman dan pengembalian buku
- Tabel data buku dan member
- Tampilan menggunakan Pico CSS (CDN)
